quickScript

QuickScript is a simple and easy to learn special-purpose language which can be used to design artificial conversational agents and other programs that involve textual dialogue between humans and computers.

Features

{easy to learn, 

easier to write}

QuickScript is special because it is simple, coherent and open source. It is intended to generate wider interest in the field of AI and virtual conversational agents, more popularly known as chatbots. The features of QuickScript are inspired by AIML, but the way of writing code has been even more simplified! QuickScript provides the basic gear that one would appreciate when coding his/her own chatbot, and it also eases the process to such an extent that anyone can playfully learn this art in minutes.
